# Strudel Journey

[Strudel REPL](https://strudel.cc/)   

#### Hot keys:
Option & Enter = play / update   
Option + . = stop   
Option + up / down key = move code up or down   
Option + Shift + up / down key = copies code up or down


#### Sounds:
sd = snare drum   
bd = bass drum   
hh = hi hat   
oh = open hat   
rim = rim   
cp = clap   

~ = rest   
[] = subdivision   
<> = selection  

.struct = ("1 0 0")   
.slow = (1)
